# teste-evertec-ponto

Backend (API):
A API pode ser compilada e executada pelo Visual Studio em modo de Debug ou Release. Todos os testes foram realizados desta forma.
O arquvio "Scripts" contém a estrutura de BD (tabela; procedures) utilizadas para no CRUD;
Desenvolvido em C# .Net Core, utilizando o padrão Restful (PontoTuristico (Get, Put, Delete, Post))

Front End:
Para executar o front, execute o comando "yarn dev" na pasta do projeto (frontend);
Desenvolvimento em TypeScript (Telas de: Home (Lista todos os pontos cadastrados); Cadastro (Realiza o cadastro); Editar(Edit do registro))
